# java-practice-date-management
顧客管理
顧客シート、物品シート、注文シートをもつexcelワークブックを作成しデータを追加したり変更したりする

■作成理由  
vscodeでもjavaを使ってみる  
■想定ユーザー  
個人事業主など  
■機能一覧  
・商品（ID）、量、価格、取引日時を記録する  
・顧客を追加  
・商品を追加  
■使用技術  
フレームワーク:VSCODE  
言語:Java  
■データ設計の考え方  
・シートごとに別のデータを記録  
・顧客シート、物品シート、注文票シートを作成  
■工夫点  
・注文票に追加するものと物品を追加するものを分けた
